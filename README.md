# 🟢 DotRescue

**DotRescue** is a minimalist, high-octane reflex game built with **Unity**. It challenges players to navigate a rotating entity through a series of dynamic obstacles. It’s a test of timing, patience, and precision.

---

## 🎮 How to Play

The mechanics are simple to learn but difficult to master:

* **PC:** Press **Spacebar** or **Left-Click** to reverse your rotation direction.
* **Mobile:** Tap the screen to switch directions.
* **Objective:** Avoid the red rotating obstacles. The longer you survive, the higher your score.
* **Difficulty Levels:** Choose your challenge from **Easy**, **Normal**, **Hard**, or the legendary **Asian** mode.

---

## ✨ Key Features

* **Dynamic Difficulty Scaling:** Obstacle speed and rotation patterns intensify based on your selected level.
* **In-game Economy:** Earn **Points** through gameplay to unlock over **20+ unique backgrounds** in the Shop.
* **Polished UX:** Features smooth score animations using `AnimationCurve` and satisfying particle effects upon collision.
* **Persistent Progress:** High scores and achievement counts (Easy-Asian completions) are saved locally using `PlayerPrefs`.
* **Cross-Platform Ready:** Optimized for both Desktop (Windows) and Mobile inputs.

---

## 🛠️ Technical Arsenal

* **Engine:** Unity 2022+ (C#)
* **UI System:** Developed with **TextMeshPro** for crisp, scalable typography.
* **Audio Architecture:** Centralized `SoundManager` using the Singleton pattern for seamless SFX management.
* **State Management:** Robust `GameManager` to handle scene transitions, scoring logic, and initialization states.

---

## 🚀 Installation (Windows)

1. Download the `DotRescue_v1.zip` file from the provided link.
2. Extract the folder to your preferred location.
3. Run `DotRescue.exe` to start the game.
   * *Note: If Windows SmartScreen appears, click **More info** -> **Run anyway**.*
